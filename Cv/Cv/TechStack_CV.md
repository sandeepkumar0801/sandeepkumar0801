## USAGE INSTRUCTIONS

### For HTML Applications:
1. **Copy the HTML code** from any format above
2. **Paste directly** into HTML editors, email templates, or web applications
3. **Customize styling** by modifying the CSS in the style attributes

### For Online CV Builders:
- Use **Format 4 (Simple Clean Table)** for basic HTML support
- Use **Format 1-3** for advanced HTML support with styling

### Color Coding in Format 3:
- **Green (#006600)**: Expert level
- **Blue (#0066cc)**: Advanced level
- **Orange (#cc6600)**: Intermediate level
